# UK Patent Filing Guide

## Overview

This guide provides instructions for filing the two patent applications with the UK Intellectual Property Office (UKIPO).

## Documents Created

1. **GB_Patent_Application_1_Spatiotemporal_Fusion.md** - Patent for hybrid SPAD-RGB object detection system
2. **GB_Patent_Application_2_Food_Safety.md** - Patent for non-invasive milk purity detection system

## Filing Requirements

### Required Documents

1. **Patent Specification** (already created)
   - Title
   - Abstract
   - Description
   - Claims
   - Drawings (references - actual drawings to be prepared)

2. **Application Form (Form 1)**
   - Available from: https://www.gov.uk/government/organisations/intellectual-property-office
   - Or file online at: https://www.ipo.gov.uk/apply-online.htm

3. **Filing Fee**
   - Online filing: £60 (reduced from £90 if filed online)
   - Paper filing: £90
   - Additional fee for claims over 25: £20 per claim over 25

4. **Drawings** (if applicable)
   - Technical drawings showing system architecture
   - Flow diagrams
   - Example results
   - Must be in black and white, clear and reproducible

### Priority Considerations

**CRITICAL:** If you plan to publish papers or present at conferences:

- **File BEFORE any public disclosure** (publication, conference presentation, thesis defense, online posting)
- UK patent law allows 12 months from first disclosure to file, but:
  - International protection (PCT) requires filing BEFORE any disclosure
  - Some countries have absolute novelty requirements (no grace period)

**Recommended Timeline:**
1. **Immediately:** File provisional applications or full applications
2. **Before Publication:** Ensure patents are filed before submitting papers to journals/conferences
3. **Within 12 months:** File international (PCT) applications if seeking global protection

## Filing Options

### Option 1: Direct UK Filing (Recommended First Step)

**Advantages:**
- Establishes priority date immediately
- Lower initial cost (£60 online)
- 12 months to decide on international filing

**Process:**
1. Complete Form 1 (Application for Grant of a Patent)
2. Submit patent specification (markdown files can be converted to PDF)
3. Pay filing fee (£60 online)
4. Receive application number and filing date

**Timeline:**
- Application number assigned immediately
- Search report: 4-6 months
- Publication: 18 months from filing
- Examination: After publication
- Grant: Typically 2-4 years from filing

### Option 2: Provisional Application (Not Available in UK)

**Note:** UK does not offer provisional applications like the US. However, you can:
- File a full application and request deferred examination
- File early and use the 12-month priority period for international filings

### Option 3: PCT International Application

**When to File:**
- Within 12 months of UK filing (to claim priority)
- If seeking protection in multiple countries

**Advantages:**
- Single application for multiple countries
- 18 months additional time to decide on specific countries
- International search and preliminary examination

**Cost:**
- PCT filing fee: ~£1,200-1,500 (including search fee)
- National phase entry: Additional fees per country

## Step-by-Step Filing Instructions

### Step 1: Prepare Final Documents

1. **Convert markdown to PDF:**
   ```bash
   # Use pandoc or similar tool
   pandoc GB_Patent_Application_1_Spatiotemporal_Fusion.md -o Patent1.pdf
   pandoc GB_Patent_Application_2_Food_Safety.md -o Patent2.pdf
   ```

2. **Prepare Drawings:**
   - Create technical drawings showing:
     - System architecture diagrams
     - Signal processing flow charts
     - Network architectures
     - Example results
   - Format: PDF, PNG, or TIFF
   - Black and white, high resolution (300 DPI minimum)

3. **Complete Application Form:**
   - Download Form 1 from UKIPO website
   - Fill in applicant details
   - List inventor(s)
   - Specify title and abstract

### Step 2: File Online (Recommended)

1. **Create Account:**
   - Visit: https://www.ipo.gov.uk/apply-online.htm
   - Register for online services

2. **Submit Application:**
   - Upload patent specification (PDF)
   - Upload drawings (if any)
   - Complete application form online
   - Pay filing fee (£60)

3. **Receive Confirmation:**
   - Application number assigned immediately
   - Filing date established
   - Receipt and confirmation email

### Step 3: Post-Filing Actions

1. **Search Request (Optional but Recommended):**
   - Request search within 12 months of filing
   - Cost: £150 (online) or £180 (paper)
   - Provides prior art search report

2. **Examination Request:**
   - Request substantive examination within 6 months of publication
   - Cost: £80 (online) or £100 (paper)
   - Examiner reviews claims and issues examination report

3. **Response to Examination:**
   - Address any objections raised
   - Amend claims if necessary
   - Respond within set deadlines

## University IP Considerations

**IMPORTANT:** If research was conducted at a university:

1. **Check Employment Agreement:**
   - University may have ownership rights
   - Review student/researcher agreement
   - Check university IP policy

2. **Contact Technology Transfer Office:**
   - University of Edinburgh Technology Transfer Office
   - They may:
     - Cover patent costs
     - Handle filing process
     - Negotiate licensing terms
     - Require assignment of rights

3. **Potential Arrangements:**
   - University files in their name (you as inventor)
   - Shared ownership agreement
   - Revenue sharing on licensing
