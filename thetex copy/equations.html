Insertion Instructions

To insert an equation in the document, use one of the following commands:

  • Unnumbered equation (display mode):
    \insertequation{EQUATION_ID}
    Example: \insertequation{1}
  • Numbered equation (appears in List of Equations):
    \insertequationnumbered{EQUATION_ID}{LABEL}
    Example: \insertequationnumbered{1}{eq:example}
  • Inline equation (within text):
    \insertequationinline{EQUATION_ID}
    Example: \insertequationinline{1}
  • Custom environment:
    \insertequationenv{EQUATION_ID}{ENVIRONMENT}
    Example: \insertequationenv{1}{align}

Available Commands

1. Unnumbered Display Equation

\insertequation{EQUATION_ID}

Inserts the equation in display mode (centered, on its own line) without a number.

2. Numbered Equation (List of Equations)

\insertequationnumbered{EQUATION_ID}{LABEL}

Inserts the equation with a number, adds it to the List of Equations, and creates a label for cross-referencing with \ref{LABEL}.

3. Inline Equation

\insertequationinline{EQUATION_ID}

Inserts the equation inline with the text (within a paragraph).

4. Custom Environment

\insertequationenv{EQUATION_ID}{ENVIRONMENT}

Inserts the equation in a custom LaTeX environment (e.g., align, gather, multline, etc.).

Technical Details

The equation system integrates with the existing List of Equations functionality in thesis.tex. When using \insertequationnumbered, equations will automatically appear in the List of Equations section.
